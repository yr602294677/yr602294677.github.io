Jar包注册为windows服务
发表于 2019-09-29| 更新于 2019-11-28|日常积累
阅读量:

  1. 从github上下载WinSW.NET4.exe，下载地址https://github.com/kohsuke/winsw/releases，然后把名字改成你想要的名字，比如demo.exe。

  2. 没有安装.net的需要先安装。

  3. 创建xml配置文件

  1. 将xml和exe文件与jar包放到同一目录下。

  2. 打开cmd，cd到jar包目录下 执行 demo.exe install安装服务，就可以在windows的服务里看到了
